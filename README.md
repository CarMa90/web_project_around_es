# Tripleten web_project_around_es

## Descripción y funcionalidad

Este proyecto es un ejemplo de una página de perfil de usuarion donde se pueden publicar fotos con sus respectivos títulos, dentro de la página se pueden modificar el nombre de usuario y descripción, además se puede eliminar, dar like o visualizar en un mayor tamaño cada tarjeta de cada imagen publicada.

[Visitar Sitio Web](https://carma90.github.io/web_project_around_es/)
